Geolocation by Using Javascript On ERPNext

This data analytics project, developed within the ERPNext system, focuses on geolocation tracking. The system prompts the user to select an employee and then specify whether the location to be recorded is primary or secondary. Upon selecting "primary," the system records the primary location, including latitude, longitude, city, and area name. Similarly, selecting "secondary" records the secondary location with the same details. Finally, the system calculates the total distance traveled between the recorded primary and secondary locations.

Google Calander Integration With ERPNext

This project, built on the ERPNext platform, integrates with Google Calendar. Leveraging an API key and client ID generated on Google Cloud Platform, the system automatically synchronizes events. When an event is created in ERPNext and synchronization is initiated, the event is automatically created in Google Calendar, and email reminders are subsequently sent for the event.
